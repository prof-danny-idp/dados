# 📚 Repositório de Dados para Avaliações

Repositório destinado ao armazenamento e organização de datasets utilizados como base para a elaboração e aplicação de provas, exercícios e atividades práticas em sala de aula.

Os dados aqui disponibilizados são utilizados em avaliações das disciplinas ministradas, podendo conter adaptações pedagógicas em relação às fontes originais.

---

## 📂 Datasets Disponíveis

### 🍷 `winequality-white.csv`

Dataset contendo características físico-químicas de amostras de vinho branco, utilizado para exercícios de análise de dados, aprendizado de máquina e estatística aplicada.

#### Atributos

| Atributo | Descrição |
|---|---|
| `fixed acidity` | Acidez fixa |
| `volatile acidity` | Acidez volátil |
| `citric acid` | Ácido cítrico |
| `residual sugar` | Açúcar residual |
| `chlorides` | Cloretos |
| `free sulfur dioxide` | Dióxido de enxofre livre |
| `total sulfur dioxide` | Dióxido de enxofre total |
| `density` | Densidade |
| `pH` | pH |
| `sulphates` | Sulfatos |
| `alcohol` | Teor alcoólico |
| `quality` | Qualidade (variável alvo, escala de 0 a 10) |

#### Referências

- Cortez, P., Cerdeira, A., Almeida, F., Matos, T., & Reis, J. (2009). *Modeling wine preferences by data mining from physicochemical properties.* Decision Support Systems, 47(4), 547-553.
- Scikit-learn documentation: https://scikit-learn.org/stable/
- Dataset: UCI Machine Learning Repository
